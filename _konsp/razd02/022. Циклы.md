# 022. Циклы
```javascript
while( условие ){}

do {} while ( условие );

for(let j=30; условие; j++){ continue; ...; } // прервать итерацию цикла

for(let j=40; условие;){ j++;}

for(let j=50;;){ ...; j++; break; } // прервать работу цикл
````

`continue` - пропустить итерацию цикла.

`break` - прервать цикл.
